# Brain Battle

An AI-powered quiz platform built using PHP and MySQL.

## Features

- User Login
- Admin Dashboard
- Quiz Categories
- Timer
- Leaderboard
- Score Analytics

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Installation

1. Clone the repository.
2. Import the SQL database.
3. Configure the database connection.
4. Run on XAMPP.

## Future Improvements

- AI-generated quizzes
